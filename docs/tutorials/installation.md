# Installation

!!! tip "Runtime type checking"
    For all installation methods, I recommend using [osyrisrblx/t](https://github.com/osyrisrblx/t). This is very useful for validating runtime types, and it is used in the tutorials.

=== "wally"
    Add the following line to your `wally.toml`:

    ```toml
    Spaceship = "wireofparadise/spaceship@1.0.0"
    ```

    And then reinstall your packages, using the following command:

    ```sh
    wally install
    ```

    !!! note "Wally package types"
        For wally, I recommend using [wally-package-types](https://github.com/JohnnyMorganz/wally-package-types) to foward exported types to the module script generated by wally. This allows you to reference Luau types used by Spaceship.

=== "roblox-ts"
    Use the following command to add Spaceship as an npm package:

    ```sh
    npm install @rbxts/spaceship
    ```

=== "rbxm"
    You can also install Spaceship as a standalone roblox model!

    Navigate to [the latest release on GitHub](https://github.com/wireofparadise/spaceship/releases/latest), download the `Spaceship.rbxm` or `Spaceship.rbxmx` file, and insert it into your place.
