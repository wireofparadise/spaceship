# Introduction

**Spaceship** is a painless & robust data store system for Roblox with schema versioning. It is very simple, exposing only one fuction for creating a collection and two methods for fetching and updating your data.

If you think that anything is missing or incorrect, feel free to open an [issue](https://github.com/wireofparadise/spaceship/issues) or submit a [pull request](https://github.com/wireofparadise/spaceship/pulls) on GitHub.
